# springboot-journal-api-basic
This is a basic Spring Boot REST API project that performs CRUD operations for a journal application.

The application allows users to:
- Create journal entries
- Retrieve all entries
- Get a specific entry by ID
- Update an entry
- Delete an entry

The project uses an in-memory data structure (HashMap) to store data, so no database is required.

This project is built for learning purposes to understand:
- Spring Boot basics
- REST API development
- Controller annotations
- Request handling (GET, POST, PUT, DELETE)
